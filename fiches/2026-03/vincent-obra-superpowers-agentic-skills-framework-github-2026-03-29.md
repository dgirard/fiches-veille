## ResumeDe400mots
Superpowers est un framework open source de skills agentiques et une methodologie de developpement logiciel creee par Jesse Vincent (obra), veteran de l'open source connu pour Request Tracker, Perl 5, Keyboardio et K-9 Mail. Avec plus de 93 000 etoiles GitHub en mars 2026, c'est le plugin le plus populaire de l'ecosysteme Claude Code, et il fonctionne desormais sur huit plateformes differentes.

La philosophie centrale de Superpowers repose sur un principe contre-intuitif : plutot que de rendre l'agent IA plus intelligent, il faut lui imposer la discipline que les developpeurs humains ont mis des decennies a construire. Le framework encode les bonnes pratiques d'ingenierie logicielle dans des "skills" composables que l'agent est contraint de suivre — ce sont des processus obligatoires, pas des suggestions.

Le workflow repose sur quatre skills fondamentaux. Le Brainstorming socratique s'active avant toute ecriture de code : l'agent affine les idees brutes par des questions, explore les alternatives et presente un document de design en sections pour validation. La Micro-planification decoupe le travail en taches de 2 a 5 minutes, chacune avec des chemins de fichiers exacts, du code complet et des etapes de verification. Le Developpement par sous-agents dispatche un agent frais par tache avec une revue en deux etapes (conformite au cahier des charges, puis qualite du code), avec des points de controle humains entre les lots. Enfin, le TDD strict impose le cycle RED-GREEN-REFACTOR : tout code ecrit avant l'existence de tests est automatiquement supprime.

Une decouverte notable de Vincent est que les principes classiques de persuasion de Robert Cialdini (autorite, reciprocite, engagement, etc.) fonctionnent sur les LLMs exactement comme sur les humains. Le framework les utilise pour maximiser la conformite des agents aux instructions. De plus, les skills se testent elles-memes via TDD : elles sont soumises a des sous-agents qui verifient qu'elles sont comprehensibles et que les agents s'y conforment.

La decision architecturale la plus significative est l'agnosticisme de plateforme. Superpowers fonctionne avec Claude Code, Cursor, Codex CLI, OpenCode, Gemini CLI, Qwen Code, Goose CLI et Auggie. Pour Claude Code, l'installation se fait en deux commandes via le marketplace officiel Anthropic.

L'ecosysteme comprend egalement superpowers-lab (skills experimentales en cours de validation), superpowers-marketplace (marketplace curee de plugins) et superpowers-chrome (controle direct du navigateur via DevTools Protocol). La contribution est ouverte : les developpeurs peuvent creer de nouvelles skills en suivant la skill dediee "writing-skills" et soumettre des pull requests.

L'impact pratique est tangible : une fois le plan approuve, l'agent peut travailler de maniere autonome pendant plusieurs heures sans devier, reproduisant le fonctionnement d'une equipe de developpement structuree.
